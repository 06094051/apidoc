# apiDoc

## About
Generates a RESTful web API Documentation.

Docs and examples at [apidocjs.com](http://apidocjs.com).

## Installation
`npm install apidoc -g`

## Example
`apidoc -i example/ -o doc/`
Creates from input files a documentation in path `doc/`.
